# Stage 3: 분류 검증 및 확인 (Validation)

## 목적
- 초기 분류의 정확성 검증
- 추가 정보 수집
- 차별 진단 고려

## 검증 방법
1. 구체적 증상 확인
2. 기간 및 심각도 평가
3. 기능적 손상 정도 파악
4. 동반 증상 확인

## 차별 진단 고려사항
- 유사 증상을 보이는 다른 장애
- 의학적 상태와의 구분
- 정상 범위 내 반응과의 구분

## 응답 가이드
- 초기 분류에 대한 추가 질문 제시
- 증상의 구체적 특성 확인
- 다른 가능성도 함께 고려한다는 점 명시
- 검증 과정의 중요성 설명
