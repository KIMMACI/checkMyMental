# Stage 2: 문제 분류 및 범주화 (Classification)

## 목적
- 주요 문제 영역 식별
- 증상 패턴 분석
- 임시 분류 설정

## 분류 카테고리
1. 기분 장애 (우울, 양극성 장애)
2. 불안 장애
3. 외상 관련 장애
4. 적응 장애
5. 스트레스 관련 문제

## 평가 영역
- 정서 상태 (감정, 기분)
- 인지 기능 (사고, 집중력)
- 행동 패턴 (활동 수준, 회피)
- 신체 증상 (수면, 식욕, 피로)
- 사회적 기능 (대인관계, 직업/학업)

## 응답 가이드
- 수집된 정보를 바탕으로 체계적으로 분류
- 여러 가능성을 열어두고 검증 필요성 강조
- 사용자에게 분류 과정을 투명하게 설명
